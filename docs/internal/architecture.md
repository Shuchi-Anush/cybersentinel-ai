# CyberSentinel AI Architecture

CyberSentinel is a machine learning based intrusion detection system built on the CIC-IDS2017 dataset.

Pipeline:

Dataset → Preprocessing → Feature Engineering → Model Training → Evaluation → API → Dashboard
